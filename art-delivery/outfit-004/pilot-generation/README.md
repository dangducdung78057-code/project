# Gate-14 Pilot Asset Generation

This directory defines the first controlled single-character pilot for `outfit-004 / Primary Dance Version / Teen Girl Center`.

## Fail-closed evidence decision

The repository currently contains Gate-13 test fixtures that assert expected PASS behavior, but it does not contain independent Approved Expansion Blueprint, Asset Generation Permission, Generation Audit Contract, or Human Approval instances for this pilot. Test fixtures are not authorization evidence. The pilot therefore remains `BLOCKED_PENDING_EVIDENCE` and no generator is invoked.

## Permitted output

The only Pilot Asset Draft in this build is a JSON metadata record. It records requested scope, frozen references, missing evidence, quality dimensions, and the block decision. It is not an image, model, garment, procurement artifact, production specification, or commercial deliverable.

## Character and age interpretation

`Teen Girl Center` is retained as the immutable existing Character Unit identifier while the target program is Primary Dance. This mismatch requires local age-proportion and stage-suitability revalidation and cannot be silently normalized or renamed.

## Required preflight chain

Approved Expansion Blueprint → Compatibility Boundary PASS → Generation Permission PASS → Generation Audit Contract PASS → Human Approval PASS → Single-character pilot authorization.

All stages are mandatory. Missing or mismatched evidence keeps the request BLOCKED. Batch generation, production handoff, procurement, and visual generation remain prohibited.
